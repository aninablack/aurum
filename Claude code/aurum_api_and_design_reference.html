What makes it feel premium
Page background is slightly warm off-white (#FAFAF8) — pure white reads clinical, warm reads expensive
Gold is used sparingly — border accents, active states, the hero strip line only. Never fill.
Cards have no shadow — elevation through border colour, not depth illusion
Data labels are 10px uppercase tracked — same language as Bloomberg, Monocle, Wallpaper*
Generous whitespace — fewer panels, breathing room between them signals confidence

Alpha Vantage — markets, FX, commodities

alphavantage.co/support/#api-key

S&P 500, DAX, FTSE, Nikkei · Gold, silver, oil · EUR/USD, GBP/USD · Free, instant

FRED — macro economic data

fred.stlouisfed.org → My Account → API Keys

CPI, Fed Funds rate, 10Y Treasury, yield curve spread · 120 req/min free · US Fed

3

NewsAPI — financial headlines

newsapi.org/register

100 req/day, search "finance OR markets OR gold OR Fed" · Classify bullish/bearish

4

metals.dev — precious metals spot prices

metals.dev → sign up (no card)

Gold, silver, platinum live from LBMA/MCX · 100 req/month (snapshot once daily = 30)

5

CoinGecko — Bitcoin + crypto as macro signal

coingecko.com/en/api → Demo API key

BTC, ETH prices + market cap · crypto Fear & Greed · 30 calls/min free

All five keys go into a .env file at the root of the repo — never committed to GitHub. Netlify reads them as environment variables at build time. The GitHub Actions pipeline reads them as repository secrets.
