## Core concept

Aurum is a financial intelligence dashboard that reads like a news product. It is not a trading terminal. It is not a data table. Every panel should feel like it was written by an intelligent analyst, not generated by an algorithm.

The UX is built around **story-driven tabs**. The user picks a lens — Overview, Geopolitical conflict, Rate hike cycle, Supply chain shock, Market correction — and the entire page content changes to tell that specific story. There are no separate "scenario overlay" buttons. The tabs ARE the scenarios.

```css

--font-ui:      'DM Sans', system-ui, sans-serif;
--font-display: 'DM Serif Display', Georgia, serif;
```

**Rules:**
- `DM Serif Display` — wordmark, briefing headline, story card headlines, asset prices, pulse values, projected impact numbers. Nowhere else.
- `DM Sans` — everything else: labels, body copy, tabs, tags, intel strips, metadata.
- Weights: 300 (rare), 400 (body), 500 (labels, deltas, active tabs). Never 600 or 700.
- All financial figures: add `font-variant-numeric: tabular-nums`
- Negative deltas: use `−` (Unicode minus U+2212), never a hyphen `-`

## DATA BINDING

Gist URL:
```js
const GIST_URL = 'https://gist.githubusercontent.com/aninablack/d1685d358f752cad1b30f28f5412c918/raw/aurum-data.json';
```

Load on page ready, refresh every 15 minutes:
```js
async function loadData() {
  try {
    const data = await fetch(GIST_URL + '?t=' + Date.now()).then(r => r.json());
    renderDashboard(data);
  } catch (err) {
    showError();
  }
}
loadData();
setInterval(loadData, 15 * 60 * 1000);
```

**Number formatting — use these helpers everywhere:**
```js
const fmt = {
  price:   n => n != null ? '$' + n.toLocaleString('en-US', { minimumFractionDigits: 2, maximumFractionDigits: 2 }) : '—',
  index:   n => n != null ? n.toLocaleString('en-US', { maximumFractionDigits: 0 }) : '—',
  pct:     n => n != null ? (n >= 0 ? '+' : '−') + Math.abs(n).toFixed(2) + '%' : '—',
  rate:    n => n != null ? n.toFixed(2) + '%' : '—',
  delta:   n => n != null ? (n >= 0 ? '+' : '−') + Math.abs(n).toFixed(2) + '%' : '—',
  abs:     (n, prefix='$') => n != null ? (n >= 0 ? '+' : '−') + prefix + Math.abs(n).toFixed(2) : '—',
  btc:     n => n != null ? '$' + Math.round(n).toLocaleString('en-US') : '—',
};
```

- No `font-weight: 600` or `700` — maximum is 500.
- No inline styles for colours that should come from CSS variables.
